## Légende

### Pinning
- **pinned** : version exacte / SHA40 / digest immuable
- **floating** : plage semver, tag mutable, branche, range ouverte
- **unpinned** : aucune version spécifiée
- **external-ref** : git+, file:, link:, workspace:, etc.

### Cooldown — réf. https://cooldowns.dev/

**Configurable côté repo applicatif** :
- npm (`min-release-age`), yarn (`npmMinimalAgeGate`), pnpm (`minimumReleaseAge`), bun (`minimumReleaseAge`), deno (`minimumDependencyAge`)
- uv (`exclude-newer`), pixi (`exclude-newer`)
- bundler (`cooldown` / `BUNDLE_COOLDOWN`)

**Non** configurable côté repo applicatif → à gérer via proxy / registre / Dependabot cooldown :
- **pip**, poetry, pdm, pipenv
- composer, cargo, nuget, maven, gradle
- github-actions, docker
